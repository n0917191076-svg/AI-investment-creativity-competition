# AI Taiwan Stock Prediction System

## Overview
This project builds a dual-engine investment system combining:
- Quantitative models (LSTM, XGBoost, RF, GB, Prophet)
- LLM-based sentiment analysis
- Market regime detection

## Key Features
- 38 engineered features
- Walk-forward backtesting
- Stacking Meta-Learner
- Risk control via regime switching

## Performance
- Cumulative Return: 205%
- Sharpe Ratio: 1.57
- Max Drawdown: -18.14%

## Models
- LSTM
- XGBoost
- Random Forest
- Gradient Boosting
- Prophet
- Stacking Meta-Learner
